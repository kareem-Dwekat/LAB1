Important:
If you are using macOS, you may encounter restrictions when installing Python packages globally due to the system’s managed environment (PEP 668).
To avoid installation errors, you must create and activate a virtual environment before installing the paho-mqtt library.
Create and activate a virtual environment:
python3 -m venv venv
source venv/bin/activate
Then install the required library:
pip install paho-mqtt
